## What is Collab-it?

Collab-it is the **team member app** of the Collab-it platform. Install it on every BIM team member's machine and connect to your organisation's [Collab-it Server](https://github.com/cr8-click/collab-it-server) over the LAN.

- Create, assign, filter, and close BIM issues
- Attach images and BCF files directly to issues
- Track the full edit and audit history of every issue
- Export to Excel, CSV, or BCF packages
- Works entirely inside your LAN — **no internet connection required** for day-to-day use

Your project data never leaves your network.

## System requirements

| | |
|---|---|
| **OS** | Windows 10 or Windows 11 (64-bit) |
| **RAM** | 2 GB minimum |
| **Disk** | 200 MB |
| **Network** | Same LAN as your organisation's Collab-it Server |

> **Note:** Your organisation must have [Collab-it Server](https://github.com/cr8-click/collab-it-server) running and a valid license before team members can sign in.

## Installation

1. Download the latest `Collab-it_x.x.x_x64-setup.exe` from the **[Releases](https://github.com/cr8-click/collab-it-client/releases/latest)** page.
2. Run the installer. Accept the UAC prompt when asked.
3. Launch **Collab-it** from the Start menu.
4. Enter the server address your administrator shared (e.g. `http://192.168.1.50:PORT`) and click **Connect**.
5. Sign in with the credentials your administrator created for you.

> **Tip:** Don't know the server address? Use the **Scan local network** button — the app will discover any Collab-it Server on your LAN automatically.

## Features

| | |
|---|---|
| **Issue management** | Create, edit, assign, and resolve BIM issues with full field support |
| **File attachments** | Upload and download images and BCF files natively |
| **Audit log** | Complete edit history on every issue |
| **Multi-project** | Switch between all projects you are invited to |
| **Export** | Export issues to Excel or CSV; download BCF packages |
| **Multilingual** | English and French — follows your Windows display language |
| **Auto-reconnect** | Automatically re-discovers the server if its IP changes on the LAN |
| **Offline resilience** | Detects connectivity loss and reconnects without restarting |
